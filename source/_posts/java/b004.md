# 一 List 相关
## 1.1 List 获取元素对象中某个属性的的集合

```java
 // List<Object>
 List<User> users = new ArrayList();
 List<Long> userIds = users 
	                   .stream()
	                   .map(User::getId)
	                   .collect(Collectors.toList())
```

## 1.2 List 获取元素符合条件的元素

```java
 List<User> users = new ArrayList<>();
List<User> collect =
			users 
				.stream()
				.filter(i -> i.getId() > 10)
				.collect(Collectors.toList());

```

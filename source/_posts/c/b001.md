# 一.使用场景

> Clion使用CMake进行项目管理
> 在使用中,无论使用TEXT()包裹中文或者是_T()包裹,都会乱码

# 二.解决办法

> 搜索得到最靠谱的解决办法:
> 在CMakeLists.txt文件中添加一个设置项

```shell
# C的解决办法
set(CMAKE_C_FLAGS "${CMAKE_C_FLAGS} -Wall -fexec-charset=GBK")
# C++ 的解决办法
set(CMAKE_CXX_FLAGS "${CMAKE_CXX_FLAGS} -Wall -fexec-charset=GBK")
```
